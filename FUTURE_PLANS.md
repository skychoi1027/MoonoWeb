# 향후 개발 계획

## 🔐 1. 보안 강화 (우선순위: 높음)

### 1.1 JWT 토큰 기반 인증
- **현재 문제점**: localStorage만 사용하여 서버에서 인증 상태 검증 불가
- **구현 내용**:
  - JWT 토큰 발급 및 검증 미들웨어 구현
  - Access Token (15분) + Refresh Token (7일) 구조
  - httpOnly 쿠키 옵션 고려
  - 토큰 만료 시 자동 갱신
- **예상 작업 시간**: 2-3일
- **필요 패키지**: `jsonwebtoken`, `cookie-parser`

### 1.2 API 인증 미들웨어
- 보호된 API 엔드포인트에 인증 미들웨어 추가
- `/api/user/*`, `/api/data/*` 등에 적용
- **예상 작업 시간**: 1일

### 1.3 CORS 설정 개선
- 프로덕션 환경에서 특정 도메인만 허용
- 환경 변수로 허용 도메인 관리
- **예상 작업 시간**: 0.5일

## 💾 2. 데이터 저장 및 관리 (우선순위: 중간)

### 2.1 궁합 결과 저장 기능
- **목적**: 사용자가 계산한 궁합 결과를 저장하고 나중에 다시 볼 수 있도록
- **구현 내용**:
  - `CompatibilityResult` 모델 생성
  - 결과 저장 API (`POST /api/compatibility/save`)
  - 저장된 결과 목록 조회 (`GET /api/compatibility/history`)
  - 결과 상세 조회 (`GET /api/compatibility/:id`)
- **데이터 구조**:
  ```javascript
  {
    userId: String,
    user1: Object,
    user2: Object,
    result: Object,  // compatibility, radarData, detailedAnalysis 등
    createdAt: Date
  }
  ```
- **예상 작업 시간**: 2-3일

### 2.2 궁합 히스토리 페이지
- 저장된 궁합 결과 목록 표시
- 날짜별, 점수별 정렬
- 결과 삭제 기능
- **예상 작업 시간**: 2일

### 2.3 즐겨찾기 기능
- 자주 확인하는 궁합 결과를 즐겨찾기로 저장
- 빠른 접근을 위한 별도 페이지
- **예상 작업 시간**: 1일
